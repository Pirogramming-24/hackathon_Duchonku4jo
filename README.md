# 🧩 피로매치 (Piro Match)

피로매치는 피로그래밍 활동 중 진행되는 **프로젝트 팀 편성을 공정하고 효율적으로 자동화**하기 위한 웹 서비스입니다.

기존에는 운영진이 성비, 실력 편차, 이전 팀 구성 이력 등을 고려하여 수기로 팀을 편성해 왔으나,

이 과정은 많은 시간과 노력이 필요하고 일관된 기준을 유지하기 어렵다는 문제가 있었습니다.

피로매치는 이러한 팀 편성 기준을 **시스템화**하여

운영 부담을 줄이고, 구성원 모두가 납득할 수 있는 팀 편성 결과를 제공하는 것을 목표로 합니다.

## ✨ 주요 기능

- **주차별 자동 팀 편성**
    - 성비, 전공/비전공 여부, 개인 과제 점수 기반 균형 배정
    - 특정 인원 간 반복적인 팀 배정을 최소화
- **팀 편성 결과 시각화**
    - 팀별 평균 과제 점수 표시
    - 팀원 성별에 따라 배경색을 달리하여 직관적 확인 가능
- **이전 팀 구성 이력 반영**
    - 이전 주차에 함께 활동한 인원이 과도하게 겹치지 않도록 자동 조정
    - 주차별 팀 편성 히스토리 관리
- **멤버 정보 관리**
    - 멤버 정보 1회 등록 후 반복 활용
    - 주차별 팀 편성 시 자동 반영

## 🧠 팀 배정 알고리즘

팀 배정은 **2단계 절차**로 진행됩니다.

### 1️⃣ 1차 배정 – 전공/비전공 및 실력 균형

1. 전체 부원을 **전공 / 비전공(O/X)** 기준으로 분리
2. 각 그룹을 **이전 주 과제 점수 내림차순 정렬**
    - 점수 동점 시 랜덤 정렬
3. 전공자 스네이크 배정
    - `1조 → 5조 → 5조 → 1조`
4. 비전공자 스네이크 배정
    - `5조 → 1조 → 1조 → 5조`

→ 전공 여부와 실력 분산을 고려한 1차 팀 구성 완료

---

### 2️⃣ 2차 배정 – 이전 팀 중복 최소화

- 이전 주차에 **같은 팀이었던 인원이 3명 이상**일 경우 조정
- 다른 팀의 **동일 전공/비전공 + 동일(또는 유사) 과제 점수** 부원과 교체
- 문제가 해소될 때까지 반복 수행

## 🖥 웹 페이지 구성

### 1️⃣ 주차 선택 페이지 (홈)

- 팀 편성을 진행할 주차 선택
- 주차 등록 및 삭제
- 멤버 관리 페이지로 이동 가능

### 2️⃣ 멤버 관리 페이지

- 전체 멤버 목록 조회 및 삭제
- 멤버 등록 페이지 제공

### 3️⃣ 팀 편성 페이지

- 선택된 주차 기준 팀 편성 알고리즘 실행
- 새로 고침을 통한 재편성 가능
- 팀 평균 과제 점수 표시
- ‘저장’ 버튼을 통해 팀 편성 확정 (재접속 시 유지)

### 4️⃣ 이전 주차 과제 점수 입력 페이지

- 멤버별 이전 주차 과제 점수 입력
- 알고리즘 반영용 데이터 저장

## 🚀 실행 방법

### 1️⃣ 가상환경 설정

```bash
python -m venv venv
# mac/linux
source venv/bin/activate
# windows
# venv\Scripts\activate
```

### 2️⃣ 패키지 설치

```bash
pip install -r requirements.txt
```

### 3️⃣ 마이그레이션

```bash
python src/manage.py migrate
```

### 4️⃣ 서버 실행

```bash
python src/manage.py runserver
```
